# covid-financial-impact-canadian-families-powerbi

## 📊 Project Overview
This repository contains a Power BI project that analyzes the financial impact of COVID-19 on Canadian households.  
It focuses on changes in income, expenses, and consumption patterns before and after the pandemic across different demographic groups.

## 🧠 Problem Statement
COVID-19 caused major financial disruptions for households in Canada, including income loss, spending changes, and economic instability.  
This project aims to analyze these changes and identify the most affected groups using data visualization.

## 📁 Dataset Features
- Family Size  
- Age Group  
- Area (Urban / Suburban / Rural)  
- Income Before COVID-19  
- Income After COVID-19  
- Expenses Before COVID-19  
- Expenses After COVID-19  

## 🧹 Data Processing
- Missing values handling  
- Duplicate removal  
- Data standardization  
- Outlier detection and cleaning  
- Balanced demographic representation  

## 📊 Visualizations
- Line Chart: Income & expenses trends over time  
- Bar Chart: Age group comparison  
- Bar Chart: Geographic impact analysis  
- Stacked Column Chart: Expense categories breakdown  

## 📈 Key Insights
- Income decreased significantly after COVID-19  
- Expenses shifted toward essential needs  
- Urban households were more affected  
- Middle-aged groups faced higher financial pressure  
- Clear change in consumption behavior
